# benchmark2_supp_data
This repository contains supplementary data for the benchmark2 repository.

The JMAB1_data folder contains the simulation and analysis results of JMAB 1.0 as a R workspace packed in a 4 part ZIP archive.

The JMAB1_simplified folder contains the simplified JMAB 1.0 model which was used to conduct experiments on the JMAB 1.0 mechanisms.
